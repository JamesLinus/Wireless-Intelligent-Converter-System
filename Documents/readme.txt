This is a Wireless Intelligent Converter Embedded System.

It used :

  Wireless Device :
  
           Texas Instruments WL1835MOD hardware device.
           
  Bluetooth module Device :
          Texas Instruments CC2564moda hardware device.
          
  Hex Buffer/Driver module Device :
  
          Texas Instruments SN74LVC07A hardware device.
          
  Board Device :
  
        
        
  MicroController :
  
           STC90C58AD series MCU(STC 8051)
           
  LCD Display Device:
  
           536 Y/JG graphic Display module.
           (http://www.lcdmozu.com/show-57-142.html)
           
  The End !
